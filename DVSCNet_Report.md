# DVSC-Net 模型说明与实验结果分析

## 一、文档目的

本文档用于补充说明新模型 **DVSC-Net** 的网络结构、设计思路与实验表现，并与 **PVSC-Net** 做简要对比。同时结合训练过程曲线图和三组混淆矩阵，对本次四分类任务的结果进行分析。

本次任务包含 4 个类别：Cargo、Passengership、Tanker、Tug。根据实验结果，DVSC-Net 与 PVSC-Net 都已经接近饱和精度，说明当前分类类别较少、类别间差异相对明显，任务难度偏低，因此很难体现出 DVSC-Net 相对于 PVSC-Net 的结构优势。

---

## 二、DVSC-Net 的整体结构

DVSC-Net 可以看作是在 PVSC-Net 基础上做出的双分支增强版本。其核心思想不是只依赖单一路径提取频谱特征，而是让网络同时从 **局部细节** 和 **上下文信息** 两个角度编码梅尔频谱，再进行特征融合和变分分类。

整体流程如下：

```text
输入梅尔频谱 (1, H, W)
    ↓
共享卷积干线 Stem
    ↓
┌───────────────────────────────┬───────────────────────────────┐
│ 局部分支 Local Branch         │ 上下文分支 Context Branch     │
│ 标准卷积提取局部纹理          │ 空洞卷积扩大感受野            │
└───────────────────────────────┴───────────────────────────────┘
    ↓                               ↓
Local Feature Head              Context Feature Head
    ↓                               ↓
局部嵌入向量                      上下文嵌入向量
    └───────────────拼接融合───────────────┘
                    ↓
                Fusion Head
                    ↓
               μ, logvar 预测
                    ↓
               重参数化采样 z
                    ↓
                分类器输出类别
```

## 三、DVSC-Net 的模块拆解

### 3.1 共享编码干线 Stem

DVSC-Net 首先通过两层卷积块对输入频谱进行下采样和初步特征提取：

- 第 1 层：1 → 32 通道，stride=2
- 第 2 层：32 → 64 通道，stride=2
- 每个卷积块都包含 Conv2d + BatchNorm2d + LeakyReLU

这一部分的作用是先把原始时频图转换成更紧凑的中层表示，为后续双分支提取共享基础特征。

### 3.2 局部分支 Local Branch

局部分支采用标准卷积堆叠：

- 64 → 128，stride=2
- 128 → 256，stride=2
- 256 → 256，stride=1

这一分支更强调频谱中的局部纹理、局部能量分布和细节结构，适合提取船舶声学信号中较稳定的局部模式。

### 3.3 上下文分支 Context Branch

上下文分支引入了空洞卷积：

- 64 → 128，stride=2，dilation=2
- 128 → 256，stride=2，dilation=2
- 再接 1 个标准卷积块

空洞卷积的优势是能够在不显著增加参数量的前提下扩大感受野，从而让模型看到更宽的时间和频率上下文。对于水下声学目标而言，这意味着模型不仅能看见局部谐波或能量峰值，还能捕捉更长时间范围内的整体时频分布趋势。

### 3.4 双分支特征提取头

每个分支末端都接一个 Feature Extraction Head，结构为：

- 1×1 卷积压缩通道
- BatchNorm2d
- GELU
- 自适应全局平均池化
- Flatten
- 全连接映射到固定维度向量
- LayerNorm
- GELU

这里的作用是把二维特征图压缩成一维嵌入向量：

- Local Head 输出局部表征向量
- Context Head 输出上下文表征向量

这两个向量分别代表对同一输入样本的两种互补理解。

### 3.5 融合头 Fusion Head

两个分支的向量拼接后进入融合头：

- 线性层：2 × branch_dim → 512
- LayerNorm
- GELU
- Dropout

融合头负责把局部信息和上下文信息整合到同一个高维语义空间，为后续变分建模提供更加稳定的输入特征。

### 3.6 变分隐变量建模

这一部分与 PVSC-Net 的核心思想保持一致：

- 使用两个线性层分别输出隐变量分布参数 μ 和 logvar
- 通过重参数化技巧采样得到潜变量 z

公式为：

$$
z = \mu + \epsilon \cdot \exp(0.5 \cdot logvar), \quad \epsilon \sim \mathcal{N}(0, I)
$$

这样做的目的，是让分类器不是直接基于单一点特征进行判别，而是基于隐空间表示进行更鲁棒的分类。

### 3.7 分类器

DVSC-Net 的分类器为：

- LayerNorm
- Dropout
- Linear: z_dim → 256
- GELU
- Dropout
- Linear: 256 → num_classes

最终输出 4 个类别的 logits。

### 3.8 频谱增强 SpecAugment

DVSC-Net 还额外加入了训练期频谱增强：

- 频率遮挡 Frequency Mask
- 时间遮挡 Time Mask

该策略只在训练阶段按一定概率启用，用于提升模型对局部缺失和扰动的鲁棒性。这一点是 PVSC-Net 中没有显式加入的。

## 四、DVSC-Net 与 PVSC-Net 的简要对比

### 4.1 共同点

DVSC-Net 与 PVSC-Net 都属于“卷积编码 + 变分隐变量 + 分类器”的思路，二者的共同目标都是：

- 从梅尔频谱中提取高层声学表征
- 学习潜在隐变量 z
- 基于隐变量完成船舶类别识别

二者都没有引入重构分支，而是直接以分类损失作为训练目标，因此训练流程相对简洁。

### 4.2 主要差异

| 对比项 | PVSC-Net | DVSC-Net |
|---|---|---|
| 编码方式 | 单分支卷积编码器 | 共享干线 + 双分支编码 |
| 分支设计 | 无 | 局部分支 + 上下文分支 |
| 感受野建模 | 常规卷积 | 常规卷积 + 空洞卷积 |
| 特征融合 | 无显式分支融合 | 双分支嵌入拼接融合 |
| 归一化与激活 | BatchNorm + LeakyReLU 为主 | BatchNorm、LayerNorm、GELU 混合 |
| 数据增强 | 无显式频谱增强 | 含 SpecAugment |
| 模型复杂度 | 较低 | 更高 |

### 4.3 理解上的差别

PVSC-Net 更像是一个结构简洁的概率卷积分类器，重点在于用隐变量对频谱特征进行压缩和分类。

DVSC-Net 则进一步强调：

- 局部时频纹理和全局上下文应分别建模
- 两种互补特征融合后再进入隐空间，理论上更适合处理复杂、模糊或存在噪声干扰的样本

因此，从设计目标来看，DVSC-Net 的表达能力强于 PVSC-Net；但表达能力更强并不必然在简单任务上带来更高精度。

## 五、实验结果汇总

### 5.1 最终验证集准确率

| 模型 | 验证准确率 |
|---|---:|
| PVSC-Net | 0.9983 |
| VesselCNN | 0.9472 |
| DVSC-Net | 0.9974 |

相对差异：

- DVSC-Net 相比 PVSC-Net：-0.0009
- DVSC-Net 相比 VesselCNN：+0.0502

从数值上看：

- PVSC-Net 与 DVSC-Net 几乎持平
- 两者都显著优于普通 CNN 基线
- DVSC-Net 没有在本次任务中体现出超越 PVSC-Net 的明显优势

## 六、训练过程曲线分析

![三模型训练曲线](model_comparison_3models.png)

### 6.1 训练损失

- PVSC-Net 的训练损失下降最快，前几轮后即迅速接近 0
- DVSC-Net 初始损失较高，但下降速度也较快，随后逐步趋稳
- VesselCNN 收敛最慢，训练损失长期高于另外两个模型

这说明 PVSC-Net 和 DVSC-Net 都具备较强的拟合能力，而普通 CNN 对当前频谱分类任务的特征提取能力较弱。

### 6.2 训练准确率

- PVSC-Net 很快达到接近 100% 的训练准确率
- DVSC-Net 训练准确率稳步提升，最终也达到较高水平，但速度略慢于 PVSC-Net
- VesselCNN 整体上升较慢，最终训练准确率明显低于前两者

这表明 DVSC-Net 虽然结构更复杂，但优化过程并不比 PVSC-Net 更轻松，在当前任务上反而需要更多轮次才能充分逼近最优状态。

### 6.3 验证准确率

- PVSC-Net 的验证准确率在训练初期就接近饱和
- DVSC-Net 的验证准确率前期波动较明显，后期逐步提升并接近 1.0
- VesselCNN 的验证准确率持续落后

这里可以看出一个重要现象：**DVSC-Net 最终性能很高，但其优势更多体现为“具备足够能力”，而不是“明显超越 PVSC-Net”。** 当前任务过于简单时，PVSC-Net 已经足够把类别边界学清楚，DVSC-Net 的额外结构无法转化成显著收益。

## 七、混淆矩阵分析

### 7.1 PVSC-Net 混淆矩阵

![PVSC-Net 混淆矩阵](confusion_matrix_pvsc.png)

PVSC-Net 的混淆矩阵几乎完全落在主对角线上：

- Cargo：930 个样本中识别正确 929 个
- Passengership：459 个样本全部识别正确
- Tanker：664 个样本中识别正确 662 个
- Tug：237 个样本全部识别正确

仅有极少量错误，主要集中在 Cargo 与 Tanker 之间。这说明 PVSC-Net 已经把 4 个类别的边界学得非常清楚。

### 7.2 VesselCNN 混淆矩阵

![VesselCNN 混淆矩阵](confusion_matrix_cnn.png)

VesselCNN 的错误明显更多：

- Cargo 会被误分到 Passengership、Tanker 和 Tug
- Passengership 与 Tanker 之间存在较多混淆
- Tanker 也会被误分到 Cargo 和 Passengership

说明简单 CNN 虽然能够完成大部分分类，但对于类别间相似的样本，判别边界不如 PVSC-Net 和 DVSC-Net 稳定。

### 7.3 DVSC-Net 混淆矩阵

![DVSC-Net 混淆矩阵](confusion_matrix_dvsc.png)

DVSC-Net 的混淆矩阵同样非常接近理想状态：

- Cargo：930 个样本全部识别正确
- Passengership：459 个样本中识别正确 457 个
- Tanker：664 个样本中识别正确 661 个
- Tug：237 个样本中识别正确 236 个

错误数量极少，整体表现与 PVSC-Net 基本处于同一水平。

### 7.4 三者对比结论

从混淆矩阵可以得到三点结论：

1. PVSC-Net 和 DVSC-Net 的分类效果都已接近满分，远好于 VesselCNN。
2. 主要混淆仍集中在 Cargo、Passengership、Tanker 这几类更相近的船型之间，但错误数已经非常少。
3. DVSC-Net 虽然引入了双分支和上下文建模，但在这个四分类任务上并没有明显减少错误到足以和 PVSC-Net 拉开差距。

## 八、为什么 DVSC-Net 没有明显优于 PVSC-Net

本次实验最值得说明的地方在于：**DVSC-Net 的结构更强，但结果并未显著超过 PVSC-Net。** 这并不意味着 DVSC-Net 设计无效，而更可能是因为当前任务本身过于容易。

### 8.1 类别数较少

当前只有 4 个类别，类别数量偏少。对高表达能力模型而言，这样的分类空间相对简单，PVSC-Net 已经足以完成判别，因此 DVSC-Net 额外的建模能力没有发挥空间。

### 8.2 类别边界较明显

从混淆矩阵看，大部分类别几乎天然可分，特别是 Tug 和部分 Cargo 样本。既然数据本身区分度较高，复杂模型就难以进一步提升已经接近 100% 的准确率。

### 8.3 指标已接近饱和

当验证准确率已经达到 99.7% 到 99.8% 时，模型之间的差异会被压缩到千分位，难以通过单次实验稳定观察到优势。此时更多结构改进可能体现在：

- 对更复杂类别集合的适应能力
- 对噪声扰动的鲁棒性
- 对更小样本类别的泛化能力

而不是体现在当前这组简单四分类指标上。

### 8.4 复杂模型不一定在简单任务上更优

DVSC-Net 引入了：

- 双分支提取
- 空洞卷积上下文建模
- 特征融合头
- 频谱遮挡增强

这些设计理论上更适合复杂数据，但也会增加优化难度。在简单任务上，结构更简单、收敛更快的 PVSC-Net 反而更容易达到略优结果。

---

## 九、总结

DVSC-Net 是在 PVSC-Net 基础上的增强模型。它通过“共享干线 + 局部分支 + 上下文分支 + 特征融合 + 变分分类”的方式，提高了特征表达的丰富性，并加入了 SpecAugment 来增强鲁棒性。从模型设计角度看，DVSC-Net 比 PVSC-Net 更完整，也更有潜力处理复杂的水下声学分类场景。

但在本次实验中：

- PVSC-Net 验证准确率为 0.9983
- DVSC-Net 验证准确率为 0.9974
- 二者几乎没有本质差距

结合训练曲线和混淆矩阵可以判断，当前任务由于分类类别太少、难度偏低、数据边界较清晰，导致 PVSC-Net 已经足够完成任务，DVSC-Net 的结构优势无法被明显体现出来。

因此，本次实验结论应表述为：

**DVSC-Net 在当前任务上验证了可行性和高精度，但未能显著超越 PVSC-Net；主要原因不是模型无效，而是任务本身过于简单，无法充分检验新模型的优势。**
