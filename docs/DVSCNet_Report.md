# DVSC-Net 模型说明与实验结果分析

## 一、文档目的

本文档用于补充说明新模型 **DVSC-Net** 的网络结构、设计思路与实验表现，并与 **PVSC-Net** 做简要对比。同时结合训练过程曲线图和三组混淆矩阵，对本次七分类任务的结果进行分析。

本次任务包含 7 个类别：Cargo、Humpback Whale、North Atlantic Right Whale、Passengership、Sei Whale、Tanker、Tug。相较此前仅包含船舶目标的设置，本次数据额外加入了 3 个海洋生物音频类别，类别规模也更加不均衡。根据实验结果，DVSC-Net 仍保持最高验证准确率，但相对于 PVSC-Net 的提升幅度较小，说明当前任务虽然更复杂，但多数大类样本仍较易区分，模型优势主要体现在边界更复杂、样本更少的类别上。

---

## 二、DVSC-Net 的整体结构

DVSC-Net 可以看作是在 PVSC-Net 基础上做出的双分支增强版本。其核心思想不是只依赖单一路径提取频谱特征，而是让网络同时从 **局部细节** 和 **上下文信息** 两个角度编码梅尔频谱，再进行特征融合和变分分类。

整体流程如下：

```text
输入梅尔频谱 (1, H, W)
    ↓
共享卷积干线 Stem
    ↓
┌───────────────────────────────┬───────────────────────────────┐
│ 局部分支 Local Branch         │ 上下文分支 Context Branch     │
│ 标准卷积提取局部纹理          │ 空洞卷积扩大感受野            │
└───────────────────────────────┴───────────────────────────────┘
    ↓                               ↓
Local Feature Head              Context Feature Head
    ↓                               ↓
局部嵌入向量                      上下文嵌入向量
    └───────────────拼接融合───────────────┘
                    ↓
                Fusion Head
                    ↓
               μ, logvar 预测
                    ↓
               重参数化采样 z
                    ↓
                分类器输出类别
```

## 三、DVSC-Net 的模块拆解

### 3.1 共享编码干线 Stem

DVSC-Net 首先通过两层卷积块对输入频谱进行下采样和初步特征提取：

- 第 1 层：1 → 32 通道，stride=2
- 第 2 层：32 → 64 通道，stride=2
- 每个卷积块都包含 Conv2d + BatchNorm2d + LeakyReLU

这一部分的作用是先把原始时频图转换成更紧凑的中层表示，为后续双分支提取共享基础特征。

### 3.2 局部分支 Local Branch

局部分支采用标准卷积堆叠：

- 64 → 128，stride=2
- 128 → 256，stride=2
- 256 → 256，stride=1

这一分支更强调频谱中的局部纹理、局部能量分布和细节结构，适合提取船舶声学信号中较稳定的局部模式。

### 3.3 上下文分支 Context Branch

上下文分支引入了空洞卷积：

- 64 → 128，stride=2，dilation=2
- 128 → 256，stride=2，dilation=2
- 再接 1 个标准卷积块

空洞卷积的优势是能够在不显著增加参数量的前提下扩大感受野，从而让模型看到更宽的时间和频率上下文。对于水下声学目标而言，这意味着模型不仅能看见局部谐波或能量峰值，还能捕捉更长时间范围内的整体时频分布趋势。

### 3.4 双分支特征提取头

每个分支末端都接一个 Feature Extraction Head，结构为：

- 1×1 卷积压缩通道
- BatchNorm2d
- GELU
- 自适应全局平均池化
- Flatten
- 全连接映射到固定维度向量
- LayerNorm
- GELU

这里的作用是把二维特征图压缩成一维嵌入向量：

- Local Head 输出局部表征向量
- Context Head 输出上下文表征向量

这两个向量分别代表对同一输入样本的两种互补理解。

### 3.5 融合头 Fusion Head

两个分支的向量拼接后进入融合头：

- 线性层：2 × branch_dim → 512
- LayerNorm
- GELU
- Dropout

融合头负责把局部信息和上下文信息整合到同一个高维语义空间，为后续变分建模提供更加稳定的输入特征。

### 3.6 变分隐变量建模

这一部分与 PVSC-Net 的核心思想保持一致：

- 使用两个线性层分别输出隐变量分布参数 μ 和 logvar
- 通过重参数化技巧采样得到潜变量 z

公式为：

$$
z = \mu + \epsilon \cdot \exp(0.5 \cdot logvar), \quad \epsilon \sim \mathcal{N}(0, I)
$$

这样做的目的，是让分类器不是直接基于单一点特征进行判别，而是基于隐空间表示进行更鲁棒的分类。

### 3.7 分类器

DVSC-Net 的分类器为：

- LayerNorm
- Dropout
- Linear: z_dim → 256
- GELU
- Dropout
- Linear: 256 → num_classes

最终输出 7 个类别的 logits。

### 3.8 频谱增强 SpecAugment

DVSC-Net 还额外加入了训练期频谱增强：

- 频率遮挡 Frequency Mask
- 时间遮挡 Time Mask

该策略只在训练阶段按一定概率启用，用于提升模型对局部缺失和扰动的鲁棒性。这一点是 PVSC-Net 中没有显式加入的。

## 四、DVSC-Net 与 PVSC-Net 的简要对比

### 4.1 共同点

DVSC-Net 与 PVSC-Net 都属于“卷积编码 + 变分隐变量 + 分类器”的思路，二者的共同目标都是：

- 从梅尔频谱中提取高层声学表征
- 学习潜在隐变量 z
- 基于隐变量完成船舶类别识别

二者都没有引入重构分支，而是直接以分类损失作为训练目标，因此训练流程相对简洁。

### 4.2 主要差异

| 对比项 | PVSC-Net | DVSC-Net |
|---|---|---|
| 编码方式 | 单分支卷积编码器 | 共享干线 + 双分支编码 |
| 分支设计 | 无 | 局部分支 + 上下文分支 |
| 感受野建模 | 常规卷积 | 常规卷积 + 空洞卷积 |
| 特征融合 | 无显式分支融合 | 双分支嵌入拼接融合 |
| 归一化与激活 | BatchNorm + LeakyReLU 为主 | BatchNorm、LayerNorm、GELU 混合 |
| 数据增强 | 无显式频谱增强 | 含 SpecAugment |
| 模型复杂度 | 较低 | 更高 |

### 4.3 理解上的差别

PVSC-Net 更像是一个结构简洁的概率卷积分类器，重点在于用隐变量对频谱特征进行压缩和分类。

DVSC-Net 则进一步强调：

- 局部时频纹理和全局上下文应分别建模
- 两种互补特征融合后再进入隐空间，理论上更适合处理复杂、模糊或存在噪声干扰的样本

因此，从设计目标来看，DVSC-Net 的表达能力强于 PVSC-Net；但表达能力更强并不一定带来大幅度的精度提升，实际收益仍取决于类别复杂度与样本分布。

## 五、实验结果汇总

### 5.1 最终验证集准确率

| 模型 | 验证准确率 |
|---|---:|
| PVSC-Net | 0.9852 |
| VesselCNN | 0.9479 |
| DVSC-Net | 0.9887 |

相对差异：

- DVSC-Net 相比 PVSC-Net：+0.0035
- DVSC-Net 相比 VesselCNN：+0.0408

从数值上看：

- DVSC-Net 在本次七分类任务中取得最高验证准确率
- PVSC-Net 与 DVSC-Net 都显著优于普通 CNN 基线
- DVSC-Net 相比 PVSC-Net 的提升有限，但在加入海洋生物类别后仍表现出更好的整体判别能力

## 六、训练过程曲线分析

![三模型训练曲线](../curve_comparison_three_models.png)

### 6.1 训练损失

- PVSC-Net 的训练损失下降最快，前几轮后即迅速接近 0
- DVSC-Net 初始损失较高，但下降速度也较快，随后逐步趋稳
- VesselCNN 收敛最慢，训练损失长期高于另外两个模型

这说明 PVSC-Net 和 DVSC-Net 都具备较强的拟合能力，而普通 CNN 对当前频谱分类任务的特征提取能力较弱。

### 6.2 训练准确率

- PVSC-Net 很快达到接近 100% 的训练准确率
- DVSC-Net 训练准确率稳步提升，最终也达到较高水平，但速度略慢于 PVSC-Net
- VesselCNN 整体上升较慢，最终训练准确率明显低于前两者

这表明 DVSC-Net 虽然结构更复杂，但优化过程并不比 PVSC-Net 更轻松，在当前任务上反而需要更多轮次才能充分逼近最优状态。

### 6.3 验证准确率

- PVSC-Net 的验证准确率在训练初期就接近饱和
- DVSC-Net 的验证准确率前期波动较明显，后期逐步提升并接近 1.0
- VesselCNN 的验证准确率持续落后

这里可以看出一个重要现象：**DVSC-Net 最终性能最高，但领先幅度并不大。** 这说明在加入海洋生物类别后，双分支与上下文建模开始产生收益，但由于大量样本仍来自较易区分的船舶类别，这种结构优势尚未被完全放大。

## 七、混淆矩阵分析

### 7.1 PVSC-Net 混淆矩阵

![PVSC-Net 混淆矩阵](../PVSCNet/matrix_confusion_PVSCNet.png)

PVSC-Net 的混淆矩阵仍然以主对角线为主，说明其在七分类任务中保持了较强的判别能力。

从结果上看：

- 船舶大类依旧是最稳定的部分
- 新加入的海洋生物类别由于样本数极少，更容易造成波动
- 误分类更可能出现在少样本类别与部分相近船舶类别之间

### 7.2 VesselCNN 混淆矩阵

![VesselCNN 混淆矩阵](../VesselCNN/matrix_confusion_VesselCNN.png)

VesselCNN 的错误仍然明显更多，尤其在类别数量增多且分布不均衡时，简单卷积模型更难兼顾大样本船舶类别与极小样本海洋生物类别。

这说明普通 CNN 虽然能够完成大部分分类，但在处理复杂类别边界和少样本类别时，判别稳定性仍明显弱于 PVSC-Net 和 DVSC-Net。

### 7.3 DVSC-Net 混淆矩阵

![DVSC-Net 混淆矩阵](../DVSCNet/matrix_confusion_DVSCNet.png)

DVSC-Net 的混淆矩阵同样非常接近理想状态，并且在整体准确率上略高于 PVSC-Net。

可以认为：

- 双分支结构对七分类任务依然有效
- 上下文分支和融合头对更复杂类别组合提供了额外帮助
- 在总体样本仍以船舶音频为主的前提下，DVSC-Net 已经体现出一定优势，但优势幅度仍较温和

### 7.4 三者对比结论

从混淆矩阵可以得到三点结论：

1. PVSC-Net 和 DVSC-Net 仍然是本任务中的主要高性能模型，且都明显优于 VesselCNN。
2. 加入海洋生物类别后，数据分布更不均衡，少样本类别带来的不稳定性开始成为影响结果的重要因素。
3. DVSC-Net 在这个七分类任务上已经取得最高验证准确率，但与 PVSC-Net 的差距仍然较小，说明结构优势开始显现但尚未完全放大。

## 八、为什么 DVSC-Net 只小幅优于 PVSC-Net

本次实验最值得说明的地方在于：**DVSC-Net 取得了最高验证准确率，但领先幅度只有 0.0035。** 这并不意味着 DVSC-Net 的结构收益有限，而更可能说明当前任务复杂度和样本分布只部分触发了它的优势。

### 8.1 类别增加了，但大样本类别仍占主导

当前已经扩展到 7 个类别，并引入了 Humpback Whale、North Atlantic Right Whale 和 Sei Whale 三个海洋生物类别。相比原先的纯船舶分类，这一设置更复杂，但从样本量上看，Cargo、Passengership、Tanker 和 Tug 仍占据绝大多数，因此整体指标仍然主要受船舶类别表现主导。

### 8.2 多数大类边界仍相对清晰

从整体结果看，多数船舶类别依然具备较明显的可分性。也就是说，尽管任务变成了七分类，但不少样本并没有形成特别困难的判别边界，因此 DVSC-Net 的额外建模能力只会转化为有限增益，而不会带来非常大的跃升。

### 8.3 提升主要体现在复杂类别与鲁棒性上

虽然最终准确率已不再像四分类时那样接近完全饱和，但模型间差距仍然不大。此时结构改进更可能体现在：

- 对更复杂类别集合的适应能力
- 对少样本类别的泛化能力
- 对局部纹理与全局上下文的联合建模能力

而不一定完全反映在单次七分类总体准确率的绝对差值上。

### 8.4 复杂模型的优势需要更困难的数据进一步放大

DVSC-Net 引入了：

- 双分支提取
- 空洞卷积上下文建模
- 特征融合头
- 频谱遮挡增强

这些设计理论上更适合复杂数据，但也会增加优化难度。在本次任务中，DVSC-Net 已经取得略优结果，不过由于类别分布高度不均衡、少样本类别占比较低，结构优势仍未完全转化为更大的数值提升。

---

## 九、总结

DVSC-Net 是在 PVSC-Net 基础上的增强模型。它通过“共享干线 + 局部分支 + 上下文分支 + 特征融合 + 变分分类”的方式，提高了特征表达的丰富性，并加入了 SpecAugment 来增强鲁棒性。从模型设计角度看，DVSC-Net 比 PVSC-Net 更完整，也更有潜力处理复杂的水下声学分类场景。

但在本次实验中：

- PVSC-Net 验证准确率为 0.9852
- VesselCNN 验证准确率为 0.9479
- DVSC-Net 验证准确率为 0.9887

结合训练曲线和混淆矩阵可以判断，当前任务在加入海洋生物类别后已经比此前更复杂，DVSC-Net 也因此表现出最高准确率；但由于大样本船舶类别仍占主导、少样本海洋生物类别数量极少，DVSC-Net 的结构优势目前只体现为小幅领先。

因此，本次实验结论应表述为：

**DVSC-Net 在当前七分类任务上取得了最高验证准确率，并验证了双分支上下文建模的有效性；但其相对 PVSC-Net 的提升仍然较小，说明当前数据设置已经能体现优势趋势，但还不足以完全放大新模型的结构收益。**
